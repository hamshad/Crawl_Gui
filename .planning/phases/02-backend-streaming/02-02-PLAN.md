# Plan 02-02: Crawl4ai Integration and SSE Generator

## Goal
Implement the background crawl function, SSE generator, and crawl4ai hooks to stream progress events.

## Tasks

<task id="1" type="auto">
<name>Implement run_crawl background function</name>
<action>Define `run_crawl(url, event_queue)` that uses `asyncio.run()` to run an async crawl function. The async function should: put `{"event": "start"}` into queue before crawling, configure crawl4ai hooks to put progress/log events into queue, put `{"event": "done", "success": true, "markdown": result.markdown}` on completion, and put `None` sentinel after done.</action>
<files>backend.py</files>
<verify>Background thread pushes events to queue; queue receives start, progress, done, None events.</verify>
<done>`run_crawl` function exists, uses asyncio.run, puts start event, configures hooks, puts done event and None sentinel.</done>
</task>

<task id="2" type="auto">
<name>Implement generate SSE generator</name>
<action>Define `generate(event_queue)` that loops `event_queue.get()`, breaks on `None` sentinel, yields `f"data: {json.dumps(item)}\n\n"` for each event, and yields `f"data: [DONE]\n\n"` after loop.</action>
<files>backend.py</files>
<verify>Generator yields properly formatted SSE data lines for each queue item.</verify>
<done>`generate` function exists, loops queue.get(), yields SSE formatted data, handles None sentinel.</done>
</task>

<task id="3" type="auto">
<name>Add crawl4ai hooks with CrawlerRunConfig</name>
<action>Create hook functions (before_goto, after_goto, on_execution_started, etc.) that put `{"event": "progress", "status": "..."}` and `{"event": "log", "msg": "...", "level": "..."}` into the event queue. Pass these hooks via `CrawlerRunConfig` to `crawler.arun()`.</action>
<files>backend.py</files>
<verify>Hooks fire during crawl and emit progress/log events to the queue.</verify>
<done>`CrawlerRunConfig` is used with hook functions that push events to event_queue.</done>
</task>
