<task type="auto">
  <name>Task 1: Install React + Vite + TypeScript and create build config</name>
  <files>electron/package.json, electron/vite.config.ts, electron/tsconfig.json, electron/tsconfig.node.json</files>
  <action>
    1. In electron/ directory, install dependencies:
       npm install react react-dom
       npm install -D vite @vitejs/plugin-react typescript @types/react @types/react-dom

    2. Create electron/vite.config.ts:
       - Import defineConfig from 'vite' and react from '@vitejs/plugin-react'
       - Set base: './' (relative paths for Electron file:// loading)
       - plugins: [react()]
       - build.outDir: 'dist'

    3. Create electron/tsconfig.json:
       - target: "ES2020", useDefineForClassFields: true, lib: ["ES2020", "DOM", "DOM.Iterable"]
       - module: "ESNext", skipLibCheck: true
       - moduleResolution: "bundler", allowImportingTsExtensions: true, isolatedModules: true
       - moduleDetection: "force", noEmit: true
       - jsx: "react-jsx"
       - strict: true, noUnusedLocals: false, noUnusedParameters: false, noFallthroughCasesInSwitch: true

    4. Create electron/tsconfig.node.json:
       - compilerOptions: { composite: true, skipLibCheck: true, module: "ESNext", moduleResolution: "bundler", allowSyntheticDefaultImports: true, strict: true }
       - include: ["vite.config.ts"]

    Note: Do NOT add Electron-specific types yet — renderer runs in Chromium, not Node.
  </action>
  <verify>cd electron && npx tsc --noEmit -p tsconfig.node.json</verify>
  <done>vite.config.ts, tsconfig.json, tsconfig.node.json exist and TypeScript compiles without errors; react and react-dom in package.json dependencies</done>
</task>

<task type="auto">
  <name>Task 2: Install and configure Tailwind CSS with shadcn dark theme tokens</name>
  <files>electron/tailwind.config.js, electron/postcss.config.js, electron/src/index.css</files>
  <action>
    1. In electron/ directory:
       npm install -D tailwindcss@3 postcss autoprefixer
       npx tailwindcss init -p

    2. Configure tailwind.config.js:
       - content: ['./index.html', './src/**/*.{js,ts,jsx,tsx}']
       - theme.extend.colors: use CSS variables for shadcn tokens:
         background/foreground, card/card-foreground, primary/primary-foreground, secondary/secondary-foreground, muted/muted-foreground, accent/accent-foreground, destructive/destructive-foreground, border, input, ring, success
       - theme.extend.fontFamily.sans: ['Quicksand', 'sans-serif']
       - theme.extend.fontFamily.mono: ['JetBrains Mono', 'monospace']
       - theme.extend.borderRadius: { lg: 'var(--radius)', md: 'calc(var(--radius) - 2px)', sm: 'calc(var(--radius) - 4px)' }

    3. Configure postcss.config.js with tailwindcss and autoprefixer plugins

    4. Create electron/src/index.css:
       - @tailwind base; @tailwind components; @tailwind utilities;
       - @layer base with CSS custom properties for dark theme:
         --background: 222 47% 11%       (dark bg ~#111827)
         --foreground: 210 40% 98%       (light text ~#f8fafc)
         --card: 222 47% 11%
         --card-foreground: 210 40% 98%
         --primary: 210 40% 98%
         --primary-foreground: 222 2% 12%
         --secondary: 217 32% 17%        (~#1e293b)
         --secondary-foreground: 210 40% 98%
         --muted: 217 32% 17%
         --muted-foreground: 215 20% 65%
         --accent: 217 32% 17%
         --accent-foreground: 210 40% 98%
         --destructive: 0 62% 45%        (red for errors)
         --destructive-foreground: 210 40% 98%
         --success: 142 76% 36%          (green for success)
         --border: 217 32% 17%
         --input: 217 32% 17%
         --ring: 212 96% 62%             (blue for focus rings)
         --radius: 0.5rem
       - @layer base: * { border-color: hsl(var(--border)) }, body { background: hsl(var(--background)); color: hsl(var(--foreground)) }

    IMPORTANT: No gradients anywhere. Use solid colors only. Do NOT install shadcn CLI — we create components manually with Tailwind classes matching shadcn patterns.
  </action>
  <verify>cd electron && npx tailwindcss --input src/index.css --output /dev/null 2>&1 | head -5</verify>
  <done>tailwind.config.js and postcss.config.js exist with correct config; src/index.css has all shadcn CSS variables and @tailwind directives; dark theme values use solid colors (no gradient utilities)</done>
</task>
